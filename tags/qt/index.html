QtPDF 的编译与试用

QtPDF 是 Qt 封装的 pdfium 模块，以 GPLv3/LGPLv3 协议分发，也可以在 QT marketplace 购买可商业版本

下载源代码

git clone <https://code.qt.io/qt/qtwebengine.git>
cd qtwebengine
git checkout 5.12.8
git submodule update --init --recursive

注意：git checkout 选择自己 qt 的版本
